# Security LMS - מערכת למידה לאבטחת מידע

מערכת למידה מקוונת המתמקדת באבטחת מידע ומודעות לסייבר.

```
src/
  ├── components/     # רכיבי React משותפים
  ├── context/       # Context API
  ├── data/         # מידע דמה וקבועים
  ├── firebase/     # הגדרות Firebase
  ├── hooks/        # Custom Hooks
  ├── pages/        # דפי האפליקציה
  └── styles/       # סגנונות CSS
```
demo:
https://security-lms.web.app/
